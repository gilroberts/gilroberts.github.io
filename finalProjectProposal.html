Final Project Proposal

For my final project I want to create an interactive shopping site where I can sell clothes from my closet and thrfited clothes I've found. I think this will be very challenging but I would really like to create something I can use in the future. It would be three pages: the home page, the shopping page, and the checkout page. I want to utilize JS for the cart aspect and CSS for the aesthetic of a modern shopping site.



I'm taking inspiration from shopping sites like Converse and Urban Outfitters
